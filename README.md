# Exam4
This project analyzes DNA sequences by extracting k-mers (substrings of length k) and tracking both their 
frequency and the nucleotide that follows each k-mer. The program reads an input text file of sequences, 
computes k-mer counts and next-character distributions, and writes the results to an output file in a 
structured format.
